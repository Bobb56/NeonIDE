# NeonIDE

## Introduction
NeonIDE is an IDE for the Neon programming language (https://neon.raphaael.fr) for TI-84 Plus CE.
It is based on the CEdit text editor (https://github.com/Michael0x18/CEdit).

## Installation
Just send on your calculator DrMono.8xv (the font for the whole app) and NEONIDE.8xp.

## Usage
The keyboard has two main modifiers: `2nd` and `alpha`.
Changing the state of the modifier `alpha` is done by pressing and releasing [alpha].
An indicator shows on top of the screen the current state of the modifier alpha.

The modifier `2nd` is enabled by pressing [2nd] and is diabled once you release [2nd].

When the modifier `2nd` is enabled, you will type the blue characters on the keyboard, and when `alpha` is enabled you will type the green characters (capital letters when alpha is ALPHA)

If no modifier is enabled, you will type the main character of the key.

You can also select text by pressing [X,T,Θ,n] while moving the cursor.

There are other key modifiers but not very useful.

The current version NeonIDE is a prototype; it is not guaranteed to be bug-free and is aimed to be improved.

Also the current version doesn't integrate the Neon interpreter; it is basically only a text editor. But in later versions Neon will be distributed as an IDE containing both the text editor and the interpreter.
